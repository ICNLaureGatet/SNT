Le Web

fichier HTML

Le langage HTML structure des pages Web à l'aide de balises.

  • des balises de type bloc après lesquels on a un retour à la ligne et un interligne;
    par exemple <h1>, <h2>, ... (pour les titres), <p> (pour les paragraphes), <div> (pour regrouper certains blocs), <ul> ou <ol> (pour les listes), <pre> (pour écrire du code informatique) ...
  • des balises de type inline où le contenu est inséré dans la ligne (souvent à l'intérieur d'une balise de type bloc) ;
    par exemple <strong> (pour mettre une partie en gras), <i> (pour mettre en italique), <a> (pour mettre un hyperlien), <span> (pour une partie d'une phrase), <code> (pour écrire du code), <img .../> (pour insérer une image), <:br/> (pour passer à la ligne dans un paragraphe) - ces deux dernières balises s'emploient cependant différemment car elles n'admettent pas de balise fermante comme les autres, ...

Un fichier HTML ne contient que le contenu. La mise en forme et le côté dynamique nécessitent d'autres lagages.

fichier CSS

Le langage CSS permet de mettre en forme notre site.
Il donne des valeurs à certains attributs de balises, de classes (deux éléments d'une même classe seront alors dans le même style) ou d'id (qui permettent d'identifier un élément unique de la page).
Par exemple :

html{
    font-family: "Comic Sans MS";
    background-color: rgb(199, 210, 221);
    padding: 5px;
}

fichiers Javascript et Php

Javascript et Php sont des langages de programmation intégrés dans une page HTML.

  • Javascript est interprété par le client (navigateur) et permet de rendre la page dynamique ;
  • Php est sur le serveur du site Web et permet d'interpréter un formulaire de saisie présent sur une page Web.
