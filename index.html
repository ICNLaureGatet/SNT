III - Créer une page Web

Voici la partie du cours sur la construction d'une page Web et la mise en forme avec le langage CSS.

Vous y découvrirez des éléments essentiels permettant de comprendre comment on construit une page Web (à l'aide d'un fichier HTML) et comment on le rend plus ergonomique (fichier CSS).
Cette partie vous montre également que le navigateur permet d'identifier les différents composants présents dans une page Web (exploration du DOM).

La partie d) du pdf vous explique le travail à faire, je vous demande juste de faire des modifications soit du côté du fichier html soit du fichier css.

Pour réaliser le travail, vous devrez vous rendre sur cette page.

  • Si vous souhaitez travailler avec NotePad++ ou Visual Studio Code :
    vous devrez récupérer les fichiers d'origine de la page (fichiers html et css) dans le fichier zip à décompresser.
    Voici une vidéo expliquant la démarche:
    • avec NotePad++
    • avec Visual Studio Code
  • Si vous souhaitez travailler sur le site repl.it :
    il faudra récupérer le code source de la page (Ctrl+U), le sélectionner entièrement puis le coller dans repl.it;
    il faudra, de plus, dans ce code source cliquer sur le lien vers le fichier CSS et copeir entièrement le contenu de ce fichier.
    Voici une vidéo expliquant la démarche:
    avec repl.it

Ce travail vous obligera à modifier les deux fichiers présents dans le .zip. Envoyez moi alors vos fichiers modifiés :

  • par mail, mais dans ce cas il faudra mettre les deux fichiers dans un fichier .zip ;
  • sur moodle, là vous glissez juste vos deux fichiers.
